# История изменений

Все значимые изменения в проекте будут документированы в этом файле.


## [1.0.3] - 2026-02-02

### Изменено
- **Интерфейс**:
    - Удален эмодзи из заголовка главного окна (теперь только "TaskMaster").

### Исправлено
- **Интерфейс**:
    - Исправлено смещение меню фильтров при перемещении главного окна (меню теперь автоматически закрывается).
    - Заблокировано перетаскивание окна за кнопку фильтров (теперь кнопка только открывает меню).
    - Добавлена кастомная кнопка фильтров, которая предотвращает перетаскивание окна на всех уровнях.


## [1.0.2] - 2026-02-02

### Добавлено
- **GUI инсталлятор**:
    - Красивый графический интерфейс с градиентным фоном и изображением приложения.
    - Возможность выбора пути установки через кнопку "Обзор...".
    - Диалог удаления программы с прогресс-баром и подробными логами.
    - Автоматический запрос прав администратора при запуске инсталлятора.
    - Документация по командам CMD для инсталлятора (INSTALLER_COMMANDS.md).
- **Архив задач**:
    - Навигатор дат в архиве задач (аналогично главному окну).
    - Фильтрация задач по дате выполнения в архиве.

### Изменено
- Каталог в AppData переименован с `ModernTaskManager` на `TaskMaster`.
- После удаления программы окно установки не появляется.
- Улучшен дизайн инсталлятора с градиентным фоном и белым контейнером.

### Исправлено
- **Совместимость с PyInstaller**:
    - Исправлена ошибка `Failed to import encodings module` на новых ПК без установленного Python.
    - Добавлены все необходимые модули стандартной библиотеки (encodings, urllib, http, ssl и др.).
    - Исправлена ошибка `base_library.zip` при проверке обновлений.
    - Улучшена стабильность запуска с помощью `bootloader-ignore-signals`.
- **Обработка ошибок**:
    - Добавлена корректная обработка ошибок PyInstaller при проверке обновлений.
    - Улучшены сообщения об ошибках для пользователя.
- **Инсталлятор**:
    - Исправлена работа кнопки "Готово" в инсталляторе (теперь корректно закрывает окно).
    - Исправлена работа кнопки "Обзор..." (выравнивание и размер).
    - Исправлен прогресс-бар (не двигается до начала установки).

### Улучшено
- **Окно уведомлений**:
    - Формат даты изменен на DD.MM.YYYY (вместо YYYY-MM-DD) для лучшей читаемости.
    - Добавлена возможность открывать задачи по клику для просмотра полной информации.
    - Уменьшен размер окна уведомлений (320x400 вместо 400x500) для более компактного отображения.
    - Уменьшен размер шрифта: заголовок задачи 10pt (Medium), дата 8pt — соответствует главному окну.
    - Кнопка "Очистить уведомления" теперь очищает список задач в окне, не закрывая его.
    - После очистки уведомления не появляются повторно до следующего запуска программы (или до появления новых просроченных задач).
    - Окно уведомлений теперь "прилипает" к иконке уведомлений и автоматически обновляет позицию при перемещении главного окна.
    - Добавлен hover-эффект для карточек задач в окне уведомлений.

## [1.0.1] - 2026-01-30

### Исправлено
- **Согласованность тем**:
    - Границы фокуса на полях ввода и выпадающих списках теперь корректно обновляются до цвета активной темы сразу.
    - Исправлена проблема, когда разделительные линии и заголовки секций сохраняли цвета из предыдущей темы.
    - Кнопки нижней панели (Справка, Настройки, Инструменты) теперь правильно отражают цвета темы (границы и состояния наведения).
- **Улучшения интерфейса**:
    - **Читаемость выпадающего списка**: Исправлены проблемы с прозрачностью в выпадающем списке Приоритет, где текст перекрывался с фоном, и обеспечена читаемость текста во всех темах.
    - **Границы всплывающих меню**: Меню "Фильтры" и "Тема" теперь автоматически корректируют свою позицию, чтобы оставаться в пределах главного окна.
    - **Запуск**: Исправлена ошибка, когда исправления стилей применялись только после переключения тем, а не при первоначальном запуске.

## [1.0.0] - 2026-01-30

### Добавлено
- **Первоначальный релиз Modern Task Manager**.
- **Основные функции**:
    - Создание задач с тремя уровнями приоритета (Высокий, Средний, Низкий).
    - Таймер в стиле Pomodoro для каждой задачи.
    - Функциональность перетаскивания для изменения порядка задач и статуса.
    - Раздел истории "Выполненные задачи".
- **Динамический дизайн**:
    - 5 встроенных цветовых тем (Синяя, Зеленая, Фиолетовая, Темная, Розовая).
    - Прозрачность окна и настройки масштабирования интерфейса.
    - Пользовательская иконка в трее и фоновый режим.
- **Умная логика**:
    - Поддержка русской множественной формы для счетчиков задач.
    - Автоматическое сворачивание/разворачивание секций.
- **Оптимизация**:
    - Высоко оптимизированный размер исполняемого файла Windows (~60MB).
    - Быстрое время запуска (<2с).
