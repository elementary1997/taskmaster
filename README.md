# Менеджер задач (Task Manager)

Простое и удобное приложение для управления вашими задачами на русском языке.

## Особенности

- **Интерфейс на русском языке** - все элементы управления и сообщения на русском
- **Приоритеты задач** - высокий, средний, низкий
- **Статус выполнения** - можно отмечать задачи как выполненные
- **Сроки выполнения** - указание даты окончания задачи
- **Поиск и фильтрация** - быстрый поиск по названию и описанию, фильтрация по приоритету и статусу
- **Статистика** - просмотр общей статистики по задачам
- **Сохранение данных** - все задачи сохраняются в JSON-файл
- **Интуитивный интерфейс** - простой и понятный интерфейс

## Требования

- Python 3.6 или выше
- Tkinter (обычно входит в стандартную поставку Python)

## Установка

1. Скачайте файлы `task_manager.py` и `run.py` в одну папку
2. Убедитесь, что у вас установлен Python 3.6+
3. Запустите приложение

## Запуск

### Способ 1: Через run.py
```bash
python run.py
```

### Способ 2: Напрямую
```bash
python task_manager.py
```

## Использование

### Добавление задачи
1. Введите название задачи
2. Добавьте описание (необязательно)
3. Выберите приоритет (высокий, средний, низкий)
4. Укажите срок выполнения в формате ДД.ММ.ГГГГ (необязательно)
5. Нажмите "Добавить задачу"

### Управление задачами
- **Редактирование** - выберите задачу и нажмите "Редактировать" или дважды кликните по задаче
- **Отметить выполненной** - выберите задачу и нажмите соответствующую кнопку
- **Удаление** - выберите задачу и нажмите "Удалить"

### Поиск и фильтрация
- Используйте поле поиска для быстрого поиска по названию и описанию
- Выберите фильтр для отображения задач определенного приоритета или статуса

### Статистика
Нажмите "Статистика" для просмотра:
- Общее количество задач
- Количество выполненных и невыполненных задач
- Распределение по приоритетам
- Процент выполнения

## Формат хранения данных

Все задачи сохраняются в файле `tasks.json` в той же директории, что и приложение. Файл имеет следующую структуру:

```json
[
  {
    "id": 1,
    "title": "Название задачи",
    "description": "Описание задачи",
    "priority": "high",
    "status": "Не выполнено",
    "due_date": "25.12.2023",
    "created": "27.01.2026 12:30"
  }
]
```

## Цветовая индикация

- **Красный фон** - задачи с высоким приоритетом
- **Зеленый фон** - выполненные задачи
- **Серый фон** - задачи с низким приоритетом

## Горячие клавиши

- **Enter** - добавление задачи (в поле ввода названия)
- **F5** - обновление списка задач
- **Delete** - удаление выбранной задачи
